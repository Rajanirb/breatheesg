# Breathe ESG Platform

An ESG analytics platform built using Django REST Framework and React.

## Features

* ESG dashboard analytics
* Scope 1, Scope 2, Scope 3 categorization
* CSV upload support
* AI sustainability recommendations
* Audit-ready backend models
* Suspicious emission record detection
* Full-stack deployment

---

## Tech Stack

### Frontend

* React
* Axios
* Recharts

### Backend

* Django
* Django REST Framework
* SQLite
